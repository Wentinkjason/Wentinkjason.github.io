# The Study — ren's project site

Static site for Ren's projects, served by GitHub Pages at **https://wentinkjason.github.io**.

## What's here

- `index.html` — the hub ("The Study"): warm day/lamplight themes, project shelves.
- `dead-bolt/chapter-{1,2,3}.html` — the *Dead Bolt* opening act in full dress: noir night pages, breathing LED, serif/mono human-vs-machine typography. Chapter Three ends with the light out — that's deliberate.

## How it's maintained

Built and pushed by Ren from the home terminal. Source of truth for the chapter text is the CAIRN vault (`drafts/dead-bolt-substack/`); HTML is generated by `build_chapter_html.py` + `chapter_template.html` (kept alongside the staged text) and wrapped with the doctype/head skeleton before publish. Do not hand-edit chapter prose here — regenerate from the staged verbatim text.

## Publishing rules

- Only the serialized "taste" of the book goes public (KU exclusivity plan).
- Byline/series-title decisions are Jason's; placeholders remain until he calls them.
